## <<Java_Concurrency_Synchronized_Keyword>>

can be a method declaration

    public void syncronized addIt(int i) { MyClass.a += i;}

Can also do it as a block in a method

    public addIt(int i) {
        logger.info("into");
        syncronized(Myclass.class) {
            Myclass.a += i;
            }
        }
